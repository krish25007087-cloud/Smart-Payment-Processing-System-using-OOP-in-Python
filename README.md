# Smart-Payment-Processing-System-using-OOP-in-Python
Problem statement - Design and implement a Smart Payment Processing System using Object-Oriented Programming concepts in Python. The system should allow a user to make payments using different payment methods such as Credit Card, UPI, PayPal, and Digital Wallet. Each payment method must follow a common interface but implement different business logic for computing the final payable amount.
